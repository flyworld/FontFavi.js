FontFavi.js
===========

A JavaScript library to generate text favicon from HTML5 canvas

### Demo

* URL : http://blog.ryanwu.me


### Installation

Simply add to your page

```
   <script src="/js/FontFav.min.js"></script>
```
### Usage

```
var favicon = new FontFavi({
        fontFamily : 'Lato',
        text : 'RW'
    });
favicon.update();
```
